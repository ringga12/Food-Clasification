Indonesia food clasificication

Program deep learning ini adalah pengenalan citra pada makanan mieayam bakso dan sate dengan menggunakan algoritma Convolutional Neural Network (CNN) yaitu salah satu jenis neural network yang bisa digunakan pada data image. CNN bisa digunakan untuk mendeteksi dan mengenali object pada sebuah image sehingga dapat diklasifikasikan sesuai dengan kelompoknya
